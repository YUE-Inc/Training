## 使用

- 左上角“苹果” - 使用偏好 - 触控板，查看手势教学
- Spotlight Search：
  - 老款 Macbook：command + space
  - 新款 Macbook：option  + space
- 软件下载（你懂的）
  - MacX
  - 麦克党
- 快捷键
  - command + shift + 3：全屏截图（默认储存至桌面）
  - command + shift + 4：选中区域截图（默认储存至桌面）
