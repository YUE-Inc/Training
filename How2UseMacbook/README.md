## 使用

- 左上角“苹果” - 使用偏好 - 触控板，查看手势教学
- Spotlight Search：
  - 老款 Macbook：command + space
  - 新款 Macbook：option  + space
- 软件下载（你懂的）
  - MacX
  - 麦克党
  - 威锋网
- 使用习惯
  - command + shift + 3：全屏截图（默认储存至桌面）
  - command + shift + 4：选中区域截图（默认储存至桌面）
  - command + w：关闭当前页面
  - command + q：退出程序
  - command + t：新建标签页/窗口
  - command + n：新建页面/文件
  - space：预览所有文件都用“空格”，不要双击
  - 所有 .iso 文件，双击打开，无需虚拟光盘
